# Ultrasonic Thickness Measurement Procedures

Surface Preparation: Remove scale, rust, paint >10 mils, or other interfering material. Smooth enough for consistent couplant contact.
Couplant: Use water-based/glycerin-based. For >150°F, use high-temp couplant. Remove after measurement.
Technique: Four readings per CML, 90° apart. Minimum value is official. Take grid readings if localized corrosion is suspected.
Velocity Settings:
- Carbon Steel: 0.2330 in/μs
- 316/304 SS: 0.2260 in/μs
- Duplex SS: 0.2290 in/μs
- Chrome-Moly Steel: 0.2330 in/μs
- Aluminum: 0.2500 in/μs (verify alloy)
