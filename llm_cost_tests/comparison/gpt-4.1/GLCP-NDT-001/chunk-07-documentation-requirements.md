# Documentation Requirements

All readings recorded electronically; manual only as backup. Data: CML ID, reading, date, time, operator ID, equipment serial. Photos required for: each CML/tag, gauge display (CRITICAL/Alert), unusual findings, inaccessible locations (with explanation), daily calibration setup. Reports: summary, scope, equipment/personnel, tabulated results, trending data, categorized findings, recommendations, appendices (raw data, photos).
